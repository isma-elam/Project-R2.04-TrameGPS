\doxysubsection{Detailed Description}
Ce fichier contient les corps des fonctions pour vérifier le format des trames GPS. 

Les fonctions ici permettent de vérifier le type de trame, le nombre de champs, le format de l\textquotesingle{}heure, la validité des coordonnées de latitude et de longitude.
