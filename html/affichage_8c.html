Detailed Description

Ce fichier contient les corps des fonctions pour l'affichage des données GPS.

Les fonctions ici permettent d'afficher un menu, de choisir la méthode de lecture des données, d'afficher les données extraites ou chargées, et de gérer les exceptions.
